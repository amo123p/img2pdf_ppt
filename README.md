# img2pdf_ppt
图片转pdf或者PPT
图片转PDF/PPT转换器
一个功能强大的桌面应用程序，支持将任意图片文件批量转换为PDF或PPT文档，提供直观的图形界面操作。

功能特性
核心功能
本程序具备强大的批量转换能力，支持同时处理多个图片文件，一键生成PDF或PPT文档。PDF转换采用PyMuPDF技术，实现真正的无损嵌入，100%保留原始图片质量，输出文件大小与原图相近。程序支持JPG、PNG、BMP、GIF、TIF、TIFF、WebP等主流图片格式，满足各种使用场景需求。同时提供灵活的排序功能，支持上移、下移、删除等操作，方便调整图片顺序。

输出选项
输出格式支持PDF文档和PPT演示文稿两种选择。PDF文档可选择无损模式、中等质量或高压缩模式，满足不同质量需求。PPT演示文稿支持空白页、图片居中、全屏填充三种布局方式。程序会自动生成输出文件名，也支持用户自定义保存路径，使用更加灵活便捷。

界面特性
图形界面基于Tkinter开发，原生Windows风格，简洁美观且无需额外依赖。提供实时预览功能，双击列表中的图片即可在默认看图软件中查看。转换过程中实时显示进度和详细日志，便于了解处理状态。程序具备完善的错误提示和异常处理机制，确保使用过程中的稳定性和可靠性。

系统要求
本程序支持Windows 10和Windows 11操作系统，其他Windows版本理论上也可正常运行。Python版本要求3.8或更高版本。系统内存建议4GB及以上。硬盘至少需要500MB可用空间，打包后的EXE文件约为150至200MB。

安装方法
源码运行
首先克隆项目到本地，进入项目目录后执行pip命令安装所有依赖。安装完成后直接运行主程序文件即可启动图形界面。具体命令如下：

git clone https://github.com/yourusername/ImageConverter.git
cd ImageConverter
pip install -r requirements.txt
python ImageConverterGUI.py
使用EXE可执行文件
下载最新版本的EXE文件（见Releases页面），双击即可运行。这是一种绿色软件形式，无需安装Python环境和任何依赖库，下载后直接可以使用。

依赖列表
项目依赖以下Python包：Pillow用于图片处理，版本要求10.0.0及以上；PyMuPDF用于PDF无损生成，版本要求1.23.0及以上；python-pptx用于PPT生成，版本要求0.6.23及以上；PyInstaller用于打包，仅在打包时需要，版本要求6.0.0及以上。

使用说明
操作步骤
第一步，点击浏览按钮选择包含图片的文件夹。第二步，通过格式过滤器筛选只显示特定类型的图片。第三步，选中图片后使用上移、下移按钮调整顺序。第四步，切换PDF或PPT输出格式并设置相关选项。第五步，点击开始转换按钮等待完成。转换过程中可以实时查看进度和日志信息。

快捷操作
双击列表中的图片可在默认看图软件中预览。Ctrl+A组合键可以全选所有图片。Shift加点击可以进行连续多选。Ctrl加点击可以进行不连续多选。

技术架构
项目结构
项目包含以下文件和目录：ImageConverterGUI.py是主程序文件，提供完整的图形界面功能；TIF2PDF.py是命令行版本，专门用于TIF格式的无损转换；requirements.txt是Python依赖列表文件；README.md是项目说明文档；dist目录用于存放打包后的EXE文件。

技术栈
GUI框架选用Tkinter结合ttk，这是Python自带的GUI库，无需额外安装。图片处理使用Pillow库，负责图片读取、尺寸获取和格式转换。PDF生成使用PyMuPDF（fitz）库，实现无损嵌入原始图片数据。PPT生成使用python-pptx库，可以创建标准的PowerPoint文件。打包工具使用PyInstaller，可以将Python程序打包为Windows可执行文件。

核心算法
PDF无损转换的核心在于使用PyMuPDF直接嵌入原始图片数据。通过调用page.insert_image方法，将图片文件的二进制数据直接嵌入PDF，不进行任何压缩处理，确保输出PDF与原始图片质量完全一致。图片列表排序使用Python内置的sorted函数进行字母排序，保证不同运行环境下的页面顺序一致。

常见问题
转换PDF后文件大小与原图相差很大？ 请确保选择无损推荐质量选项。默认使用PyMuPDF进行无损嵌入，文件大小应与原始TIF文件总大小相近。如果选择其他压缩选项，文件大小会明显减小，但图片质量也会相应降低。

支持哪些图片格式？ 程序支持JPG、JPEG、PNG、BMP、GIF、TIF、TIFF、WebP、ICO、PPM等常见图片格式。

EXE文件杀毒软件报毒？ PyInstaller打包的程序有时会被杀毒软件误报。建议将EXE添加到杀毒软件信任区，或者直接使用源码运行程序。

可以转换WebP格式吗？ 可以。程序原生支持WebP格式转换，PDF输出采用无损模式。

生成的PPT图片模糊？ PPT使用的是原始图片数据，不会出现模糊情况。可能需要在PowerPoint中调整图片大小和位置以获得最佳显示效果。
